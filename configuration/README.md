## Default Configuration Files

Note: Do not alter these files unless you know what you are doing.

### action_filters.json

Contains a list of animation action names to skip during loading.

### armature_links.json

This functionality is currently disabled in the main branch.

Contains the configurations for linking armatures using copy transforms from a source bone to a target bone.

### driver_template.json

Contains the default settings for the property groups.

### prefixes.json

Contains the prefixes added to object names.

### user_settings.json

Contains settings for the different node types as well as global settings.
